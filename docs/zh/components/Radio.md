# Radio

## 何时使用

- 用于在多个备选项中选中单个状态。
- 和 Select 的区别是，Radio 所有选项默认可见，方便用户在比较中选择，因此选项不宜过多。
- 在打开插件时需要保留上一次操作的值时，该组件可以提供基本能力。

::: tip
该组件完全支持AntDV的a-radio属性
:::
