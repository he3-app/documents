# Switch

## 何时使用

- 需要表示开关状态/两种状态之间的切换时；
- 和 checkbox 的区别是，切换 switch 会直接触发状态改变，而 checkbox 一般用于状态标记，需要和提交操作配合。
- 在打开插件时需要保留上一次操作的值时，该组件可以提供基本能力。

::: tip
该组件完全支持AntDV的a-switch属性
:::
