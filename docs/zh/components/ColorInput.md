# ColorInput

## 何时使用

在有颜色输入时，该颜色输入框可提供颜色的Input框和颜色选择器

## 代码演示

:::demo 输入色号或点击色块选择颜色
color/input
:::

## API

### Props

| 属性              | 说明     | 类型   | 默认值 |
| ----------------- | -------- | ------ | ------ |
| v-model:pureColor | 颜色绑定 | string |        |
