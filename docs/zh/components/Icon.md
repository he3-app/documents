# Icon

## 何时使用

在有需要使用图标时，该组件可以提供各种icon支持,配合[icones](https://icones.js.org/)使用

## 代码演示

:::demo 不同大小的SVG图标
icon/basic
:::

## API

### Props

| 属性 | 说明                                      | 类型   | 默认值 |
| ---- | ----------------------------------------- | ------ | ------ |
| icon | [icones](https://icones.js.org/)上的图标字段 | string |        |
| size | 图标大小                                  | number | 14     |
